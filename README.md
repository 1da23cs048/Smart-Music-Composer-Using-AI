# Smart Music Composer Using AI

## Project Description
Smart Music Composer Using AI is a web application that generates music automatically using artificial intelligence. 
Users can generate melodies and listen to AI-created compositions.

## Features
- AI-based music generation
- Web interface for generating music
- Play and download generated music
- Simple user interface

## Technologies Used
- Next.js
- Node.js
- JavaScript
- AI music generation libraries

## Installation
Clone the repository:

git clone https://github.com/1da23cs048/Smart-Music-Composer-Using-AI.git

Install dependencies:

npm install

Run the project:

npm run dev

Open in browser:

http://localhost:3000
